# Mon Portfolio Web

Bienvenue sur le dépôt de mon portfolio personnel.

Ce site présente mes projets, mes compétences, mon parcours ainsi que mes coordonnées. Il a été réalisé en HTML, CSS, JavaScript, et utilise divers frameworks front-end pour l'amélioration de l'interface utilisateur.

## 🔧 Technologies utilisées

- HTML5 / CSS
- JavaScript
- Bootstrap
- Git & GitHub

## 📁 Structure du projet

- `index.html` : page d’accueil
- `assets/` : images, CSS, JS, polices, etc.
- `forms/` : formulaire de contact
- `portfolio-details.html` : détails des projets
- `service-details.html` : détails des services
